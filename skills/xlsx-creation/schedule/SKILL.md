# Создание расписания занятий

## Контекст
Расписание занятий — основной организационно-методический документ, определяющий порядок проведения учебных занятий по дисциплинам для конкретной группы/потока в течение семестра. Составляется на основе учебного плана и рабочих программ дисциплин.

## Шаги

1. **Собрать исходные данные**
   - Группа(ы), курс, семестр, учебный год
   - Учебный план: перечень дисциплин с указанием часов (лекции, практические, лабораторные)
   - Преподаватели: ФИО, должность, ставка, часы нагрузки по дисциплинам
   - Аудиторный фонд: номера аудиторий, вместимость, тип (лекционная, компьютерный класс, лаборатория)
   - Ограничения: окна в расписании преподавателей, совмещение, дни, когда занятия невозможны
   - Формат: дневное/вечернее, очное/заочное

2. **Определить временную сетку**
   - Стандартные пары: 1-я (8:30–10:05), 2-я (10:20–11:55), 3-я (12:10–13:45), 4-я (14:15–15:50), 5-я (16:05–17:40), 6-я (17:50–19:25)
   - Учебные дни: понедельник–суббота (в зависимости от формы обучения)
   - Начало и конец семестра, каникулы, сессия
   - Распределить общее количество часов по неделям семестра

3. **Распределить дисциплины по неделям**
   - Лекции: равномерно по семестру (например, 1 раз в неделю)
   - Практические/лабораторные: с учётом тематического плана РПД
   - Консультации перед экзаменами: 2–3 недели до сессии
   - Зачёты/экзамены: последние 2–3 недели семестра (сессионная неделя)
   - Проверить: сумма часов по расписанию = сумме часов в учебном плане

4. **Составить таблицу расписания**
   - **Структура**: дни недели × номера пар
   - **В каждой ячейке**: дисциплина, тип занятия (лек./пр./лаб.), преподаватель (ФИО или инициалы), аудитория
   - **Цветовая кодировка**:
     - Лекции — один цвет
     - Практические — другой
     - Лабораторные — третий
     - Зачёты/экзамены — отдельный цвет
   - **Отдельные листы**: для каждой группы или для каждой недели (при необходимости)

5. **Проверить ограничения и сохранить**
   - Нет накладок: один преподаватель/аудитория в одно время
   - Нет «окон» у студентов более 2 пар подряд без перерыва (если требуется)
   - Преподаватель не превышает установленную нагрузку
   - Аудитория соответствует типу занятия (лаборатория для лабораторных)
   - Сохранить как: `Расписание_<Группа>_<Семестр>_<Учебный_год>.xlsx`
   - Дополнительно: экспорт в `.pdf` для публикации на стенде/сайте

## Правила

- Одна аудитория не может быть назначена на две пары одновременно
- Преподаватель не может вести две пары одновременно (даже в разных группах)
- Лабораторные занятия проводятся только в специализированных аудиториях
- Количество пар в день для группы не должно превышать 6 (рекомендуется не более 4 для очной формы)
- Физическая культура не назначается на 1-ю или 6-ю пару (по возможности)
- Зачёты и экзамены не назначаются на одну дату с лекциями по той же дисциплине
- Расписание должно быть согласовано с заведующим кафедрой и методистом деканата

## Пример

**Запрос пользователя:** «Составь расписание для группы ПИ-301, 5 семестр, очная форма»

**Действия агента:**
1. Получает данные:
   - Дисциплины: Базы данных (лек 36, лаб 36), Веб-разработка (лек 18, пр 36), Проектный практикум (пр 72)
   - Преподаватели: Иванов И.И. (лекции БД), Петров П.П. (лабораторные БД, Веб), Сидоров С.С. (практикум)
   - Аудитории: 101 (лекц., 60 мест), 202 (комп., 20 мест), 305 (комп., 20 мест)
2. Распределяет по 18 неделям:
   - БД лекции: 1 пара, пн, недели 1–18, ауд. 101, Иванов И.И.
   - БД лабораторные: 3 пара, ср, недели 1–12 (по 3 часа), ауд. 202/305, Петров П.П.
   - Веб лекции: 2 пара, вт, недели 1–9, ауд. 101, Петров П.П.
   - Веб практические: 3 пара, пт, недели 1–12, ауд. 202, Петров П.П.
   - Практикум: 2–4 пары, чт, недели 1–18, ауд. 305, Сидоров С.С.
3. Проверяет: Петров П.П. не пересекается (вт лекция + ср/пт практика = ок)
4. Красит ячейки: лекции — жёлтый, лабораторные — зелёный, практика — голубой
5. Сохраняет: `Расписание_ПИ-301_5сем_2025-2026.xlsx`

## Проверка качества

- [ ] Все дисциплины из учебного плана включены в расписание
- [ ] Сумма часов по расписанию равна учебному плану
- [ ] Нет накладок преподавателей и аудиторий
- [ ] Тип аудитории соответствует типу занятия
- [ ] Преподаватель не превышает ставку/нагрузку
- [ ] Цветовая кодировка типов занятий применена
- [ ] Расписание охватывает весь семестр до сессии
- [ ] Зачёты/экзамены вынесены в сессионную неделю
- [ ] Файл сохранён с корректным именем
