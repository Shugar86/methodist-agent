# Извлечение текста из PDF

## Контекст
PDF-документы часто используются для хранения и передачи методических материалов, учебных пособий, нормативных документов (ФГОС, приказы, стандарты). Агент должен уметь извлекать текст для анализа, цитирования, переработки или проверки на соответствие требованиям.

## Шаги

1. **Определить тип PDF и задачу**
   - Запросить у пользователя: путь к файлу или URL
   - Уточнить цель: полное извлечение, извлечение раздела, поиск конкретной информации, таблиц
   - Проверить тип PDF: текстовый (создан из Word), отсканированный (растровое изображение), смешанный
   - Для отсканированных PDF уточнить необходимость OCR (распознавания текста)

2. **Извлечь текст**
   - **Текстовый PDF**: использовать библиотеку `PyPDF2`, `pdfplumber` или `pymupdf`
   - **Отсканированный PDF**: использовать OCR (`pytesseract` + `pdf2image`)
   - **С таблицами**: использовать `pdfplumber` или `camelot-py` для структурированного извлечения
   - **Многостраничный**: извлекать постранично с сохранением нумерации
   - **Защищённый PDF**: запросить пароль у пользователя или уточнить правовой статус

3. **Структурировать извлечённый текст**
   - Сохранить иерархию заголовков (если распознаваема)
   - Выделить цитаты, формулы, таблицы отдельными блоками
   - Для таблиц: преобразовать в формат `.csv` или `.xlsx`
   - Для изображений: извлечь отдельно с указанием страницы
   - Сохранить метаданные: автор, название, дата создания, количество страниц

4. **Проверить качество извлечения**
   - Сравнить объём извлечённого текста с количеством страниц (признак ошибки: 0 символов на текстовом PDF)
   - Проверить распознавание спецсимволов (математические формулы, греческие буквы)
   - Для OCR: проверить частые ошибки («о» вместо «0», «rn» вместо «m»)
   - Проверить целостность таблиц (не съехали столбцы, не пропали строки)
   - Для многостраничных: проверить порядок страниц

5. **Сохранить результаты**
   - Полный текст: `.txt` или `.md` с именем `<Исходный_файл>_text.txt`
   - Структурированный: `.docx` с сохранением форматирования
   - Таблицы: `.xlsx` или `.csv`
   - Метаданные: JSON-файл или заголовок в начале текста
   - При необходимости: создать краткое резюме (summary) извлечённого текста

## Правила

- Не пытаться извлечь текст из защищённых авторским правом документов без разрешения
- Для OCR указывать язык документа (русский, английский, смешанный) — это критично для качества
- При извлечении из ФГОС и нормативных документов сохранять точность формулировок — цитаты должны быть дословными
- Таблицы с числовыми данными извлекать с максимальной точностью (десятичные знаки, единицы измерения)
- Если качество OCR низкое (менее 90% распознавания), предупредить пользователя и предложить ручную проверку
- Не удалять переносы слов при извлечении — сохранять оригинальную пунктуацию для цитирования
- Для больших документов (100+ страниц) предлагать извлечение по разделам или оглавлению

## Пример

**Запрос пользователя:** «Извлеки текст из файла `ФГОС_09.03.04.pdf`, мне нужны компетенции»

**Действия агента:**
1. Определяет тип: текстовый PDF, 15 страниц
2. Извлекает полный текст через `pdfplumber`
3. Ищет раздел «Планируемые результаты обучения» и «Компетенции обучающегося»
4. Структурирует:
   - Универсальные компетенции (УК-1…УК-8)
   - Общепрофессиональные компетенции (ОПК-1…ОПК-4)
   - Профессиональные компетенции (ПК-1…ПК-12)
5. Сохраняет:
   - Полный текст: `ФГОС_09.03.04_text.txt`
   - Таблица компетенций: `ФГОС_09.03.04_компетенции.xlsx` (столбцы: Код, Название, Описание)
6. Проверяет: сравнивает извлечённые коды компетенций с оглавлением — совпадают

## Проверка качества

- [ ] Тип PDF определён корректно (текстовый/отсканированный)
- [ ] Для OCR указан правильный язык
- [ ] Объём извлечённого текста соответствует ожидаемому
- [ ] Спецсимволы и формулы распознаны корректно
- [ ] Таблицы структурированы без потери данных
- [ ] Нумерация страниц сохранена
- [ ] Метаданные документа зафиксированы
- [ ] Цитаты дословно соответствуют оригиналу
- [ ] Результаты сохранены в удобном для дальнейшей работы формате
- [ ] Пользователь предупреждён о возможных ошибках OCR
