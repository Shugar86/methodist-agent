# Создание методического отчёта

## Контекст
Методический отчёт — документ, отражающий результаты методической работы кафедры, преподавателя или методиста за определённый период. Используется для отчётности перед деканатом, методическим советом или аккредитационной комиссией.

## Шаги

1. **Уточнить тип и период отчёта**
   - Запросить у пользователя: за какой период (семестр, учебный год, 5 лет)
   - Тип отчёта: индивидуальный (преподавателя), кафедральный, факультетский
   - Цель отчёта: внутренняя отчётность, аккредитация, аттестация кафедры
   - Объём: краткий (2–3 стр.) или полный (10–20 стр.)

2. **Собрать исходные данные**
   - Список преподавателей (для кафедрального отчёта)
   - Перечень дисциплин, по которым велась методическая работа
   - Разработанные/обновлённые документы (РПД, методические указания, тесты)
   - Результаты опросов студентов, анализ успеваемости
   - Участие в методических семинарах, конференциях, курсах повышения квалификации
   - Внедрение новых технологий, цифровых инструментов, дистанционных форм

3. **Сформировать структуру отчёта**
   - **Титульный лист**: наименование ОО, кафедра, период, должность составителя
   - **Введение**: цель и задачи методической работы, актуальность
   - **Раздел 1. Анализ состояния преподавания**: успеваемость, качество знаний, проблемные темы
   - **Раздел 2. Методические разработки**: перечень созданных/обновлённых документов с датами
   - **Раздел 3. Инновационная деятельность**: новые технологии, цифровизация, кейсы
   - **Раздел 4. Результаты и эффективность**: количественные показатели (часы, документы, охват студентов)
   - **Заключение**: выводы, планы на следующий период
   - **Приложения**: таблицы, графики, копии документов (при необходимости)

4. **Написать содержание**
   - Использовать конкретные цифры и факты, избегать общих фраз
   - Для каждой методической разработки указывать: название, авторов, дату, статус (новая/обновлённая)
   - В аналитических разделах приводить сравнение с предыдущим периодом
   - Планы на будущий период должны быть измеримыми (конкретные дисциплины, сроки)

5. **Оформить и сохранить**
   - Шрифт Times New Roman 14, межстрочный 1.5, поля 2 см
   - Заголовки разделов: полужирный, 16 пт
   - Таблицы: границы видимые, заголовки повторяются на новых страницах
   - Нумерация страниц внизу по центру
   - Сохранить как: `Методический_отчёт_<Кафедра>_<Период>.docx`

## Правила

- Каждое утверждение должно подкрепляться данными или ссылкой на документ
- Не использовать оценочные суждения без доказательной базы («всё было отлично»)
- Указывать реальные даты и конкретные дисциплины, а не абстрактные формулировки
- Для кафедрального отчёта охватить всех преподавателей, а не только активных
- Планы на следующий период должны быть реалистичными и согласованными с учебным планом
- Соблюдать требования к антиплагиату: оригинальность текста не ниже 70%

## Пример

**Запрос пользователя:** «Составь методический отчёт кафедры „Информационные системы“ за 2025/2026 учебный год»

**Действия агента:**
1. Запрашивает: состав кафедры (5 преподавателей), основные направления работы
2. Собирает данные:
   - Обновлено 4 РПД (Программирование, Базы данных, Веб-разработка, Проектный практикум)
   - Разработано 2 методических указания к лабораторным работам
   - Проведён опрос 120 студентов, средняя оценка 4.2/5
   - Успеваемость: 87% на отлично и хорошо (рост на 5% к прошлому году)
   - 3 преподавателя прошли курсы повышения квалификации
3. Формирует структуру отчёта (8 страниц)
4. В разделе «Анализ» приводит таблицу сравнения успеваемости по семестрам
5. В заключении формулирует планы: обновить 3 РПД под ФГОС 3++ (срок — I семестр 2026/2027)
6. Сохраняет: `Методический_отчёт_ИС_2025-2026.docx`

## Проверка качества

- [ ] Указан корректный период отчётности
- [ ] Все разделы структуры заполнены
- [ ] Приведены конкретные цифры и факты
- [ ] Упомянуты все преподаватели (для кафедрального отчёта)
- [ ] Есть сравнение с предыдущим периодом
- [ ] Планы на будущее конкретны и измеримы
- [ ] Текст оригинален (нет шаблонных фраз из интернета)
- [ ] Форматирование соответствует стандарту вуза
- [ ] Файл сохранён с корректным именем
