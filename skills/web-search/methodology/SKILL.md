# Поиск методических материалов

## Контекст
Методические материалы — учебники, учебные пособия, методические указания, кейсы, тесты, презентации, используемые в образовательном процессе. Агент помогает методисту найти актуальные, достоверные и легально доступные материалы по заданной теме или дисциплине.

## Шаги

1. **Уточнить запрос пользователя**
   - Тема или дисциплина (например, «методические указания к лабораторным работам по БД»)
   - Целевой уровень: бакалавриат, магистратура, СПО
   - Тип материала: учебник, пособие, методичка, тесты, презентации, видеолекции
   - Язык: русский, английский, bilingual
   - Ограничения: только бесплатные, только официальные издательства, только с ISBN
   - Год издания: не старше N лет (рекомендуется для технических дисциплин — не старше 5 лет)

2. **Сформировать поисковые запросы**
   - **Русскоязычные источники**:
     - ЭБС (электронно-библиотечные системы): «Лань», «Юрайт», «КонсультантСтудент», «ibooks.ru»
     - Федеральные порталы: «РИОР», «Федеральный центр информационно-образовательных ресурсов»
     - Репозитории вузов: eLIBRARY.ru, КиберЛенинка, репозитории конкретных университетов
   - **Международные источники**:
     - Google Scholar, ResearchGate, Academia.edu
     - Открытые учебники: OpenStax, MIT OpenCourseWare, Coursera/edX materials
     - Документация и спецификации: IEEE, ACM, W3C, ISO
   - **Поисковые операторы**:
     - `filetype:pdf` для поиска PDF-документов
     - `site:edu.ru` для ограничения доменом
     - `intitle:методические указания` для поиска в заголовке

3. **Выполнить поиск и отфильтровать результаты**
   - Проверить актуальность: год издания, версия стандарта (например, ФГОС 3++ vs ФГОС 3+)
   - Проверить достоверность: издательство (ДМК Пресс, Питер, БХВ-Петербург, официальные вузовские издательства)
   - Проверить легальность: наличие на легальных ЭБС, открытых лицензий (Creative Commons), статуса «доступно на КиберЛенинке»
   - Исключить: сомнительные сайты с рекламой, файлообменники без указания источника, материалы с водяными знаками пиратских библиотек

4. **Структурировать найденные материалы**
   - Для каждого источника указать:
     - Название, автор(ы), год издания
     - Издательство или источник
     - Тип: учебник / учебное пособие / методические указания / статья / видео
     - Объём (страницы, минуты)
     - Ссылка (прямая или на каталог)
     - Примечание: платно/бесплатно, требуется регистрация вуза, открытый доступ
   - Сгруппировать по типу материала
   - Отметить рекомендуемые источники (наиболее полные и актуальные)

5. **Представить результаты пользователю**
   - Сформировать список в формате `.md` или `.docx`
   - Для каждого материала дать краткую аннотацию (2–3 предложения)
   - Указать, какие материалы подходят для каких целей (лекции, практика, самостоятельная работа)
   - При наличии: приложить ссылки на скачивание или просмотр
   - Сохранить как: `Методические_материалы_<Тема>_<Дата>.md`

## Правила

- Приоритет отдавать материалам, рекомендованным к изданию в установленном порядке (гриф УМО, гриф НИИ образования)
- Для дисциплин по программированию и IT рекомендовать материалы не старше 3–5 лет (технологии устаревают быстро)
- Для гуманитарных дисциплин допустимы материалы 10–15-летней давности (классика)
- Не рекомендовать материалы с пиратских библиотек (Z-Library и аналоги) — только легальные источники
- Указывать, если материал доступен только при наличии подписки вуза (ЭБС «Лань», «Юрайт»)
- Для ФГОС 3++ отдавать предпочтение материалам с компетентностным подходом, а не просто знаниевым
- Проверять, что найденные материалы соответствуют уровню подготовки (не рекомендовать магистерские материалы бакалаврам без адаптации)

## Пример

**Запрос пользователя:** «Найди методические материалы по дисциплине „Машинное обучение“ для бакалавров 3 курса»

**Действия агента:**
1. Уточняет: русский язык, учебники и пособия, не старше 2020 года
2. Выполняет поиск:
   - ЭБС «Лань»: «Машинное обучение» — находит 5 учебников
   - КиберЛенинка: находит 3 статьи и 1 пособие
   - MIT OpenCourseWare: находит курс 6.867 с лекциями и заданиями
3. Фильтрует:
   - Исключает: книгу 2015 года (устаревшие библиотеки)
   - Отмечает: учебник с грифом УМО (приоритет)
   - Помечает: материалы MIT — на английском, требуют перевода или bilingual группу
4. Формирует список:
   1. «Машинное обучение: учебное пособие» — Авторов А.А., 2023, «Лань», 320 стр., гриф УМО, доступно по подписке вуза
   2. «Введение в машинное обучение с Python» — Müller A.C., Guido S., 2021, ДМК Пресс, 400 стр., платно
   3. Лекции MIT 6.867 — бесплатно, английский, с заданиями на Python
5. Сохраняет: `Методические_материалы_МО_2026-01-15.md`

## Проверка качества

- [ ] Запрос уточнён (уровень, тип материала, язык, год)
- [ ] Поиск выполнен по нескольким источникам (не менее 3)
- [ ] Результаты отфильтрованы по актуальности и достоверности
- [ ] Исключены сомнительные и пиратские источники
- [ ] Для каждого материала указаны: автор, год, издательство, тип, доступность
- [ ] Материалы сгруппированы по типу и отсортированы по релевантности
- [ ] Рекомендованы наиболее подходящие материалы с обоснованием
- [ ] Указаны ограничения доступа (подписка, язык)
- [ ] Результаты сохранены в структурированном формате
